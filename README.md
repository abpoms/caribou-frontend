# Caribou Frontend

A Page hierarchy and routing to action mapping based on Caribou Core model structure.

## Usage

<pre>
> lein new caribou tundra
> cd tundra
> lein caribou bootstrap resources/config/development.clj
> lein run
</pre>

Then navigate to http://localhost:33333
